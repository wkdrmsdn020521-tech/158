# 방송용 마블 룰렛 프로토타입

Matter.js 기반으로 구현된 방송용 핀볼 룰렛 HTML 데모입니다. 하나의 HTML 페이지와 연동된 `styles.css`, `app.js`만으로 구성되어 있어 정적 호스팅이나 로컬 환경 어디서든 실행할 수 있습니다.

## 실행 방법

1. **코드 다운로드**
   - Git으로 클론하거나, ZIP으로 내려받아 원하는 폴더에 압축을 풉니다.
2. **로컬에서 열기**
   - `index.html` 파일을 웹 브라우저(Chrome, Edge, Firefox 등)로 직접 열면 즉시 실행됩니다.
   - 또는 `python -m http.server 8000` 같은 간단한 정적 서버를 띄운 뒤 `http://localhost:8000/index.html`로 접속해도 됩니다. (방송 PC에서 OBS 브라우저 소스로 띄울 때 추천)
3. **리소스 확인**
   - `index.html`이 `styles.css`와 `app.js`를 상대 경로로 불러오므로 세 파일을 같은 디렉터리에 둡니다.

## 기본 사용법

1. **참가자 등록**
   - 왼쪽 상단 "참가자 구성" 텍스트 박스에 이름을 입력합니다.
   - 쉼표, 줄바꿈으로 여러 참가자를 한 번에 입력할 수 있습니다.
   - `이름*숫자` 형태(예: `장근우*2`)로 적으면 해당 이름의 핀볼이 곱한 수만큼 생성됩니다.
2. **핀볼 생성**
   - 입력을 마쳤다면 `핀볼 생성` 버튼을 누릅니다. 즉시 필드에 공이 등장하고 게임이 시작됩니다.
   - 이미 진행 중인 핀볼이 있다면 버튼을 다시 눌러 추가로 투입할 수 있습니다.
3. **섞기 / 초기화**
   - `섞기` 버튼은 현재 필드에 있는 핀볼 위치를 무작위로 뒤섞어 변수를 늘려줍니다.
   - `초기화` 버튼은 필드의 모든 공과 순위 데이터를 삭제하고 대기 상태로 되돌립니다.
4. **진행 상황 보기**
   - 중앙 필드에서 실시간으로 핀볼이 떨어지고 장애물과 상호작용하는 모습을 볼 수 있습니다.
   - 왼쪽 "로드맵" 패널은 전체 맵을 축소해서 보여주며 마우스 오버 시 시야가 이동합니다.
   - 오른쪽 "실시간 순위" 패널에서 골인지점까지의 거리 순으로 순위를 확인할 수 있고, 골인을 통과한 참가자는 이름 왼쪽에 체크 표시가 붙습니다.
   - 남은 핀볼이 2~3개가 되면 자동으로 카메라가 확대되고 속도가 느려집니다.

## 방송 적용 팁

- **OBS 브라우저 소스**
  - `index.html`을 호스팅한 주소를 OBS 브라우저 소스로 추가하면 바로 송출할 수 있습니다.
  - 1920×1080 해상도에 맞춰 제작되어 있으므로 브라우저 소스 크기를 동일하게 맞추면 가장 자연스럽습니다.
- **참가자 템플릿**
  - 자주 쓰는 참가자 명단을 메모장에 저장해두고 필요할 때 붙여넣으면 빠르게 세팅할 수 있습니다.
- **디버그**
  - 개발자 도구(Chrome 기준 F12)를 열면 콘솔에 물리 엔진 관련 로그가 출력됩니다. 버그가 의심될 때 확인해 보세요.

## 파일 구성

- `index.html` – UI 레이아웃과 패널 정의
- `styles.css` – 네온 테마 스타일 및 반응형 레이아웃
- `app.js` – Matter.js 기반 게임 로직, 참가자 파서, 카메라 & 연출 제어

필요에 따라 텍스트/이미지를 커스터마이즈해 자신의 방송 스타일에 맞춰 활용하세요.
